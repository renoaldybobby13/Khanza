# üçè Diet Tracker PWA

Aplikasi **Progressive Web App (PWA)** untuk melacak diet harian, kalori, dan berat badan. Bisa digunakan **offline** dan dipasang seperti aplikasi di HP.

---

## Fitur

1. **Profil Pengguna**
   - Input Nama, Jenis Kelamin, Umur, Tinggi, Berat, Target Berat
   - Kalkulasi **BMR** otomatis
   - Hitung **kalori harian yang dianjurkan**
   - Tampilkan **jumlah kunjungan** ke aplikasi

2. **Log Makanan**
   - Tab Sarapan, Siang, Sore, Malam
   - **Preset makanan + input manual** nama dan kalori
   - Tambah, edit, hapus log makanan
   - Hitung **total kalori** & sisa kalori untuk hari itu

3. **Motivasi**
   - **Motivasi harian** tetap sama sepanjang hari
   - **Motivasi anti-malas** acak, berubah otomatis setiap 10 detik

4. **Grafik**
   - **Grafik kalori 7 hari terakhir**
   - **Grafik berat badan** mingguan

5. **Push Notification**
   - Pengingat input makanan setiap hari (opsional, browser mendukung)

6. **Offline Support**
   - Menggunakan **Service Worker**, dapat dibuka tanpa internet
   - Bisa di-install sebagai PWA di HP atau desktop
